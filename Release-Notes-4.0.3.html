ATS : Release Notes 4.0.3

22 December 2017

New Features

 Amazon S3 Support operations over Amazon Simple Storage Service (S3)

Here and here are dedicated pages for S3 operations and S3 verification respectively.


REST Client Support multi-value headers

Use the addRequestHeader method as many times as needed to add one more value to the pointed header.

As opposite, the setRequestHeader method will replace an existing header value.

More about our RestClient utility.


Logging system Log a console warning when having different versions between Test Executor and ATS Agent

This warning is sometimes a good indication of possible issues when working with the ATS Agent.
