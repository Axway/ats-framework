ATS : Logging test method javadoc

Created by Nikolay Gogovski, last modified on Mar 27, 2017

Introduction

By default test method javadoc is not logged. You can use it as a scenario description.

All you need to do is to set a system variable with name  ATS_SOURCE_FOLDER_LOCATION to point to the folder containing your test class sources.

 

How to do it

There are two ways:

  • Run the following code before the execution of your first test
System.setProperty( "ATS_SOURCE_FOLDER_LOCATION", "D:\\WorkFolder\\WorkProject\\src" );
  • Add the following to the start command of the process running your tests:
-DATS_SOURCE_FOLDER_LOCATION="D:\\WorkFolder\\WorkProject\\src"
